Command Lists

1) Enter Configuration Mode#configure terminal

2) Enter Interface Mode #interface fa0/1

3) Enter HostName #hostname switch1

4) Enter Banner Message #banner motd $

5) Select One Inferface #int fa0/1

6) Select Interface Range #int range fa0/1-11

7) Enter description #description _ _ _ _

8) Add vlan using config mode #vlan 3

9) Set auto timezone #clock summer-time est 0

10) Set clock rate#clock rate 64000

11) Enable no domain lookup #no ip domain-lookup

12) Display History #show history

13) Display Interface status#show ip int brief

14) Display Protocals #show ip protocals

15) Display IP routing table show ip route
