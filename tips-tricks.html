10 Tips & Tricks for Beginners

1) You can practice routing/switching using Cisco Packet Tracker online. Click Here for the Link to Download

2) If you need help Cisco Community is a great place to look for help.

3) You can use Up Key [↑] Arrow to go back to previous command.

4) You can use Down Key Arrow [↓] to go back to new command line.

5) Use Copy & Paste tool provided in command window to save time.

6) Use [TAB] key to complete commands.*

7) Type [?] (question mark) whenever you feel confused because it provides list with aviliable commands you can use.

8) Press [CTRL] + [F6], if stucked in loading window to quit.

9) To download commands Press File and Download from main menu

10) Practice!!! Practice!!! Practice!!!


Getting Started with Packet Tracer

1) Login with your User Name and Password.

2) Bottom left corner click Router and drag it to white space.

3) Next to Router, click Switches and drag it to the white space.

4) Click on End Devices and select PC then drag that to white space.

5) Click the thunder bolt sign and Select Solid Black cable.

6) Go back to white space, now click on Router and select G0/0/0.

7) Drag that cable to Switch and Press on switch then slect F0/1.

8) Repeat step 5.

9) Click on PC and Select F0/0.

10) Drag that cable to Switch and Press on switch then slect F0/2.
